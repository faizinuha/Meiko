Perkakas Baris Perintah (CLI)

Kelola proyek Laravel Anda langsung dari terminal.

coders-solid

Menyiapkan proyek Laravel dengan struktur SOLID dan konfigurasi awal yang rapi.

coders-crud

Membuat file CRUD dasar untuk proyek Laravel Anda dengan satu perintah sederhana.

coders-delete

Menghapus folder atau file proyek Laravel dengan aman jika tidak lagi dibutuhkan.
